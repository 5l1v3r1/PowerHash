# PowerHash
Optimization Crypto algorithms.


Algorithms
-------

- **Blake256** (crypto/blake256.c)
- **Groestl** (crypto/groestl.c)
- **Jh** (crypto/jh.c)
- **Keccak** (crypto/keccak.c)
- **Skein** (crypto/skein.c)

Compile and Building
-------------------
```
$ mkdir _build
$ cd _build
$ cmake ..
$ sudo make -j8
$ sudo make install
$ ./PowerHash
```
